ArrayList与LinkedList的区别

发表于 2017-06-07 | 分类于 java

结构区别

  • ArrayList 的实现是基于动态数组的数据结构。
  • LinkedList 的实现是基于链表的数据结构。

使用场景

  • 对于随机的get 或者 set 访问ArrayList 优于LinkedList 因为LinkedList 需要移动指针。
  • 对于新增或者删除操作（add 或者 remove）ListedList 效率优于ArrayList 因为ArrayList 需要移动数据。
